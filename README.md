# Automação - Dias Sem Acidentes
Script em Python para atualizar automaticamente um docx de dias sem acidentes.

O que ele faz:

-Atualiza a data atual
-Calcula dias corridos desde uma data base
-Atualiza valores no arquivo .docx
-Converte para PDF
-Envia para impressão

Utilizado em conjunto do agendador de tarefas do Windows
